# Гистограм

Веб-приложение для анализа и коррекции изображений. Позволяет загрузить изображение, просмотреть его гистограммы (общую яркость и отдельно по каналам R, G, B), настроить экспозицию и баланс белого, а затем скачать результат.

## Возможности

- Загрузка изображений через drag & drop или выбор файла (JPEG, PNG, WebP)
- Гистограммы в стиле Photoshop — 4 мини-графика: яркость (L), красный (R), зелёный (G), синий (B)
- Настройка экспозиции — яркость и контраст с возможностью автокоррекции
- Настройка баланса белого — температура и оттенок с возможностью автокоррекции
- Неразрушающее редактирование — все коррекции всегда применяются к оригиналу
- Скачивание результата в JPEG

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Приложение запустится на http://127.0.0.1:5000

## Технологии

- **Backend:** Python, Flask, OpenCV, NumPy
- **Frontend:** HTML, CSS, JavaScript, Chart.js
