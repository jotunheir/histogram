# Гистограм

Веб-приложение для анализа и цветокоррекции изображений — яркость, контраст и баланс белого с мгновенным предпросмотром гистограмм в стиле Photoshop.

---

## Возможности

**Анализ**
- 4 гистограммы в реальном времени: яркость (L), красный (R), зелёный (G), синий (B)
- Гистограммы обновляются при каждом изменении параметров

**Экспозиция**
- Яркость — гамма-коррекция, сдвигает средние тона без жёсткого клиппинга светов и теней
- Контраст — S-образная кривая вокруг средне-серого

**Баланс белого**
- Температура — сдвиг между холодными и тёплыми тонами (синий ↔ красный)
- Оттенок — коррекция зелёного ↔ пурпурного канала

**Автокоррекция**
- Авто-экспозиция — вычисляет оптимальную гамму по среднему и подбирает контраст по ширине гистограммы (2–98 перцентиль)
- Авто-баланс белого — выравнивает каналы по среднему значению

**Общее**
- Drag & drop или выбор файла (JPEG, PNG, WebP)
- Неразрушающее редактирование — коррекции всегда применяются к оригиналу
- Скачивание результата в JPEG (92% качество)
- Адаптивный интерфейс — двухколоночный на десктопе, одноколоночный на мобильных

## Как работает обработка

```
Загрузка (JPEG/PNG/WebP)
  → Декодирование (OpenCV, BGR)
    → Яркость: гамма-коррекция через LUT (gamma = 2^(-b/100))
      → Контраст: степенная S-кривая через LUT
        → Температура/Оттенок: поканальная арифметика (float32)
          → Кодирование в JPEG (92%) → base64 → клиент
```
